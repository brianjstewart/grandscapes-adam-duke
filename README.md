# GrandScapes Website

Professional landscape design and installation website for GrandScapes (Adam Duke) - Mount Pleasant, SC.

## Features

✅ **Light Mode Design** - Clean, professional aesthetic
✅ **Real Portfolio Images** - 12 before/after project examples
✅ **Responsive Layout** - Mobile-first design
✅ **Fast Performance** - Next.js with optimized images
✅ **Contact Form** - Lead generation built-in
✅ **Trust Signals** - Awards, credentials, testimonials

## Pages

1. **Homepage** - Hero section with trust signals and featured projects
2. **Services** - Design, Installation, Maintenance, Hardscape with details
3. **Portfolio** - 12 before/after projects with filtering by category
4. **About** - Company story, team, credentials, recognition
5. **Contact** - Contact form with phone, email, location info

## Build & Deployment

### Development

```bash
npm install
npm run dev
```

Visit `http://localhost:3000`

### Production Build

```bash
npm run build
npm start
```

### Deploy to Vercel

1. Push to GitHub: `grandscapes-adam-duke` repo
2. Vercel auto-deploys from main branch
3. Live at: https://grandscapes-adam-duke.vercel.app

## Technology Stack

- **Next.js 15** - React framework
- **Tailwind CSS** - Utility-first styling
- **Lucide Icons** - SVG icons
- **TypeScript** - Type safety
